## ✨ 프로젝트 소개

- 말랑을 통해 문해력을 키워보세요.
- 학습 퀴즈를 통해서 문해력을 학습해보세요.
- 학습한 단어와 틀린 단어는 단어장과 오답노트에 기록됩니다.
- 본인의 문해력으로 점령지들을 점령하고, 팀에 승리에 기여해보세요.
